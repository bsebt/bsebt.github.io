Hardware Development

Magic Paper

Simple Explanation:

Increasing security by using each person's capacitance in their body and a pattern. Using these two unique indentifier we can make safer locks.

Technical Explanation:

Magic paper was a prototype made in Mac Master University Hackathon (Delta Hacks). One of my teammate knew that each person has a different capacitance and using this knowledge we decided to make a better security system which can identify people using their body capacitance and a unique pattern. In this prototype, we used 4 aluminum plates to measure, and calibrate each person's capacitance and follow the pattern they move their finger in order to identify themselves.

Java Application development

Guitar Tablature

Simple Explanation:

An application used to convert text-based guitar tablature into a customizable PDF document.

Technical Explanation:

This project was made for a class called "Software Development Project". I can confidently say that it has been my favourite class of university so far.
With 3 others, we were to use a java library called iText to take an input of guitar text manuscripts (in ASCII format) and convert it into formatted PDF guitar tabs in the duration of 4 months.
A Decision was made to make the most simple user interface in order to use this application, however, this course was about complexity of design, so we decided to convert to PDF and design a second screen to allow users to customize their PDF files and save. Using all the tools available in the Java swing library, we were able to design a simple but useful program.
In terms of testing, we used JUnit to challenge and test our program for different inputs to make sure we do not have any unhandled errors in the system.
