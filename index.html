Java Apllication development

Guitar tablature

Simple Explonation:

An application used to convert text-based guitar tablature into a customizable PDF document.

Technical Explonation:

This project was made for a class called "Software Development Project". So far I can say it was my favorite course of university.
We were given 4-months, group of 4-people, and a java library called iText in order to take an input of guitar text manuscript(in ASCII format) and convert it into formatted PDF guitar bars.
Decision was made to make the most simple user interface in order to use this application, however, this course was about complexity of design, so we decided to convert to PDF and design a second screen to allow users to customize their PDF version and save. Using all the tools available in java swing library, we were able to design a simple but usful program.
In terms of testing, we used Junit testing to challenge and test our program for any input and make sure we do not have any unhandled errors in the sytem.
